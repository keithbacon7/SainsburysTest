
Install this project to directory  [project_root]

	To run 
		CD [project_root]
		src\Run.bat



To build and run tests requires a machine with a recent version of Maven. 


Command Window commands
	To run tests and build project
		CD [project_root]
		mvn clean package

	To re-run tests
		CD [project_root]
		mvn test
